# UI/UX 设计指南

## 设计原则

### 用户体验原则
- **简洁高效**: 界面简洁，操作流程最短化
- **一致性**: 保持整个应用的视觉和交互一致性
- **可访问性**: 支持键盘导航，符合 WCAG 2.1 AA 标准
- **响应式**: 适配桌面、平板、手机等不同设备
- **容错性**: 提供清晰的错误提示和恢复机制
- **组件库**: 非必要时不要自定义类，使用成熟的组件库

### 药房专业需求
- **快速操作**: 扫码和数据录入要快速响应
- **清晰展示**: 药品信息、批次、有效期等关键信息突出显示
- **安全确认**: 重要操作需要二次确认
- **实时反馈**: 库存变化、操作结果实时更新

## 组件设计规范

### 扫码界面
- 扫码区域占据屏幕中央，边框清晰可见
- 扫码成功后立即显示药品信息
- 提供手动输入条码的备选方案
- 扫码历史记录便于快速重复操作

### 表单设计
- 必填字段用红色星号标记
- 实时验证，错误信息在字段下方显示
- 使用合适的输入类型（数字、日期、文本）
- 提供输入建议和自动完成功能

### 数据表格
- 关键信息（药品名称、库存数量）使用较大字体
- 近效期药品用橙色/红色标记
- 库存不足用醒目颜色提醒
- 支持排序、筛选、搜索功能

### 通知和提醒
- 成功操作用绿色提示
- 警告信息用橙色提示
- 错误信息用红色提示
- 重要提醒支持声音通知（可配置）

## 颜色系统

### 主色调
- **主色**: #2563eb (蓝色) - 用于主要按钮和链接
- **辅助色**: #10b981 (绿色) - 用于成功状态
- **警告色**: #f59e0b (橙色) - 用于警告信息
- **错误色**: #ef4444 (红色) - 用于错误状态
- **中性色**: #6b7280 (灰色) - 用于次要文本

### 语义化颜色
- **库存充足**: #10b981 (绿色)
- **库存不足**: #f59e0b (橙色)
- **库存为零**: #ef4444 (红色)
- **近效期**: #f59e0b (橙色)
- **已过期**: #ef4444 (红色)

## 布局规范

### 页面布局
- 使用侧边栏导航，主内容区域最大化
- 顶部显示用户信息和快速操作
- 底部显示版本信息和帮助链接
- 移动端使用底部导航栏

### 响应式断点
- **手机**: < 640px
- **平板**: 640px - 1024px
- **桌面**: > 1024px

### 间距系统
- **基础间距**: 4px 的倍数 (4, 8, 12, 16, 20, 24, 32, 48, 64)
- **组件内间距**: 8px - 16px
- **组件间间距**: 16px - 24px
- **页面边距**: 24px - 32px

## 交互规范

### 按钮设计
- **主要按钮**: 填充样式，用于主要操作
- **次要按钮**: 边框样式，用于次要操作
- **危险按钮**: 红色填充，用于删除等危险操作
- **按钮状态**: 正常、悬停、激活、禁用

### 加载状态
- 数据加载时显示骨架屏或加载动画
- 长时间操作显示进度条
- 按钮点击后显示加载状态，防止重复提交

### 操作反馈
- 成功操作显示 Toast 提示
- 错误操作显示详细错误信息
- 重要操作需要确认对话框

## 移动端优化

### 触摸友好
- 按钮最小尺寸 44px × 44px
- 增加触摸目标间距
- 支持手势操作（滑动、长按）

### 扫码优化
- 全屏扫码界面
- 支持手电筒功能
- 自动对焦和连续扫码

### 输入优化
- 数字输入调用数字键盘
- 日期输入使用日期选择器
- 支持语音输入（如适用）

## 可访问性规范

### 键盘导航
- 所有交互元素支持键盘访问
- 清晰的焦点指示器
- 合理的 Tab 顺序

### 屏幕阅读器
- 使用语义化 HTML 标签
- 提供 alt 文本和 aria-label
- 重要状态变化要通知屏幕阅读器

### 视觉辅助
- 支持高对比度模式
- 文字大小可调节
- 不依赖颜色传达信息

## 性能优化

### 图片优化
- 使用 WebP 格式
- 实现懒加载
- 提供不同尺寸的图片

### 动画优化
- 使用 CSS 动画而非 JavaScript
- 支持减少动画偏好设置
- 避免影响性能的复杂动画

## 错误处理界面

### 404 页面
- 友好的错误提示
- 提供返回首页的链接
- 搜索功能帮助用户找到内容

### 网络错误
- 显示网络连接状态
- 提供重试按钮
- 离线模式提示

### 表单错误
- 字段级别的错误提示
- 表单提交失败的整体提示
- 提供修正建议
