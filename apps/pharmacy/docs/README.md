# 文档目录

本目录包含药房库存管理系统的所有文档。

## 目录结构

### 设置和配置文档

- `AUTHENTICATION_SETUP.md` - 认证系统配置指南
- `DATABASE_SETUP.md` - 数据库设置指南
- `SUPABASE_SETUP.md` - Supabase 配置指南
- `ENVIRONMENT.md` - 环境变量配置
- `TESTING.md` - 测试指南
- `SEED_DATA_README.md` - 种子数据说明

### 示例代码

- `examples/` - 代码示例和最佳实践
  - `env-usage-example.ts` - 环境变量使用示例

### 优化指南

- `AUTH_OPTIMIZATION_GUIDE.md` - 认证系统优化指南
- `COMPONENT_OPTIMIZATION_GUIDE.md` - 组件性能优化指南

### 归档文档

- `archive/` - 已完成的修复指南和历史文档
  - `fixes/` - 修复相关文档
  - `optimization/` - 优化相关文档
  - `deployment/` - 部署相关文档

## 使用说明

1. **新用户**：先阅读根目录的 `README.md`，然后查看设置文档
2. **开发者**：参考 `examples/` 目录中的代码示例
3. **运维人员**：查看部署和配置相关文档
4. **历史参考**：查看 `archive/` 目录中的历史文档

## 文档维护

- 活跃的文档保留在主目录
- 完成的修复指南移至 `archive/`
- 代码示例放在 `examples/`
- 定期清理过时文档
