# 红医师基层辅助用药系统

## 项目简介
基层医疗辅助诊断系统，用于辅助基层门诊医生和卫生员在面对常见临床症状时，合理选择药品和处理方式。

## 技术栈
- Vite
- React
- Tailwind CSS
- shadcn-ui
- Capacitor (Android)

## 本地开发
确保已安装 Node.js & npm，然后执行：

```bash
git clone <repository-url>
cd care-symptom-assistant
npm install
npm run dev
```
## 部署
项目使用 Vercel 进行部署，支持自动化部署流程。

### 部署步骤
1. Fork 本仓库到你的 GitHub 账号
2. 在 Vercel 中导入该项目
3. 配置环境变量（如需要）
4. 部署完成后即可访问
