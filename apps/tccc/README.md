# TCCC 战术战斗伤员救护交互式流程

这是一个基于CoTCCC 2017版官方算法流程图开发的交互式战场急救决策辅助系统。项目旨在以可视化、交互式的方式，帮助用户理解和应用标准的战术战斗伤员救护流程。

## 项目特点

- **完全匹配CoTCCC指南**: 基于2017年1月31日版本的官方CoTCCC指南实现
- **交互式决策流程**: 用户可以进行选择，系统会根据决策引导用户进入下一步
- **能力标识系统**: 明确标注不同流程需要的人员能力等级
  - 绿色: 所有作战人员和战斗生命救护员能力等级
  - 黄色: 战斗医护人员能力等级
  - 红色: 战斗医疗队或特种作战医护人员能力等级
- **多语言支持**: 中英文双语显示，便于国际合作训练
- **响应式设计**: 适配各种设备，从手机到桌面显示器
- **视觉引导**: 使用醒目的红色高亮关键步骤和行动点

## 已实现流程

1. **TCCC标准流程**: 基础战术战斗伤员救护流程，覆盖交火中救治到战术区域救治
2. **TCCC与伤员沟通**: 战术区域救治中的沟通流程，包括与伤员、战术指挥链和医疗后送链的沟通
3. **TFC大出血算法**: 大出血控制与处理流程，包括止血带、止血敷料应用等
4. **TFC气道算法**: 气道评估与管理流程，包括基础气道管理和高级气道技术，按照2017版CoTCCC指南实现
5. **TCCC呼吸管理算法**: 呼吸评估与管理流程，包括张力性气胸处理与开放性胸部伤口管理
6. **TCCC休克与液体复苏**: 循环评估与管理流程，休克识别与液体复苏处理
7. **TCCC静脉通路与止血酸**: 静脉通路建立与止血酸使用流程，包括骨髓内通路备选方案
8. **TCCC伤口处理**: 战场伤口处理流程，包括烧伤管理与创伤包扎
9. **TCCC骨盆绑带流程**: 骨盆骨折评估与绑带使用流程
10. **TCCC疼痛管理**: 战场疼痛评估与药物镇痛流程
11. **TCCC创伤性脑损伤**: 脑损伤评估与管理流程
12. **TCCC预防低体温**: 战场低体温预防与管理流程
13. **TACEVAC再评估**: 战术后送过程中的伤员再评估流程，包括出血再评估和干预措施检查

## 如何使用

1. 直接在浏览器中打开 `index.html` 文件
2. 通过索引页访问各个流程
3. 按照页面上的指引，点击按钮进行决策和流程导航
4. 注意观察每个步骤右上角的能力等级标识，确保实际操作人员具备相应能力

## 技术实现

- 纯前端实现，无需后端服务器
- 使用HTML5、CSS3和JavaScript ES6+开发
- 使用TailwindCSS进行样式设计
- 使用Intersection Observer API实现页面滚动动画效果
- 使用Font Awesome图标库提供视觉元素

## 关于TCCC

战术战斗伤员救护（TCCC）是一套基于循证医学的战场伤员救治标准，由美国国防部制定并持续更新。TCCC把战场救护分为三个阶段：交火中救治、战术野战救治和战术后送，针对不同战术环境下的救护提供了明确的指南。
