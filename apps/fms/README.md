# FMS 功能性动作筛查系统

本项目是一个完整的FMS（Functional Movement Screen）功能性动作筛查评估系统，旨在帮助用户进行科学的动作模式评估，识别潜在的功能性限制和伤病风险。

## 🌟 核心功能

### 📋 完整的FMS评估体系

**基础动作测试（7项）**
1. 深蹲 (Deep Squat) - 评估全身协调性和灵活性
2. 跨栏步 (Hurdle Step) - 评估单腿稳定性和分离运动能力
3. 直线弓步 (Inline Lunge) - 评估下肢柔韧性和控制能力
4. 肩部灵活性 (Shoulder Mobility) - 评估肩关节活动范围
5. 主动直腿抬高 (Active Straight-Leg Raise) - 评估腘绳肌柔韧性
6. 躯干稳定俯卧撑 (Trunk Stability Push-up) - 评估核心稳定性
7. 旋转稳定性 (Rotary Stability) - 评估多平面核心控制

**排除性测试（3项）** ⚠️
1. 肩部撞击测试 (Shoulder Impingement Clearance) - 排除肩部病理性问题
2. 脊柱屈曲测试 (Spinal Flexion Clearance) - 排除脊柱屈曲方向问题  
3. 脊柱伸展测试 (Spinal Extension Clearance) - 排除脊柱伸展方向问题

### 🎯 智能评估功能

- **智能悬浮进度条**：页面顶部悬浮显示，滚动时智能隐藏，停止时重新显示
  - 实时显示测试进度（X/10）
  - 当前测试名称和类型提示
  - 完成百分比和剩余项目统计

- **动态评分系统**：支持0-3分评分，排除测试采用通过/未通过机制

## 📖 使用指南

### 开始评估
1. 访问主页，点击"开始FMS评估"
2. 系统将引导您完成10项测试（7项基础 + 3项排除）
3. 根据演示视频和评分标准为每个动作评分

### 理解评分
**基础测试评分（0-3分）：**
- **3分**：动作完美，无代偿
- **2分**：动作完成但有轻微代偿
- **1分**：无法完成标准动作
- **0分**：测试中出现疼痛

**排除测试评分：**
- **通过**：无疼痛或不适
- **未通过**：出现疼痛，需要专业评估

### 查看报告
- **总分分析**：基础测试总分（满分21分）
- **风险评估**：14分以下提示需要改善
- **问题识别**：自动标识需要关注的动作模式
- **专业建议**：基于评分结果的个性化建议

## ⚠️ 重要安全提示

### 疼痛处理原则
如果任何测试中出现疼痛（0分），请：
1. **立即停止**相关动作测试
2. **不要进行**纠正性训练
3. **咨询专业医疗人员**（物理治疗师或医生）
4. **获得许可后**再继续运动

### 排除测试重要性
排除测试用于识别可能需要医疗干预的问题：
- 肩部撞击综合征
- 脊柱功能障碍
- 其他病理性问题

**未通过排除测试时应立即寻求专业医疗评估**
